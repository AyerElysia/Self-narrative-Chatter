   新增统一日志管理，支持日志轮转、自动压缩、历史查询，可在设置中配置保留天数和文件大小。

   全新安装向导，分离显示 MoFox 和 NapCat 状态，优化性能和交互体验。


Neo-MoFox 核心更新

集成 WebUI 插件 (4096074)

重要：已有旧版 WebUI 的用户请先删除再拉取 dev 分支

功能说明：
将 MoFox-Core-Webui 完整集成为插件（112 文件，11000+ 行代码），提供以下功能：
- 仪表盘、实时日志、日志查看器、聊天室管理
- Bot/Model/Plugin 配置管理
- 插件市场和插件管理
- Git 更新、UI 更新、环境管理
- 主题配置、关系图谱、表情管理、模型统计
技术支持：FastAPI、WebSocket、组件化架构
